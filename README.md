# MoeICP-CLI

萌国ICP备案第三方CLI

# 安装
```bash
pip3 install requests lxml
git clone https://github.com/2096779623/MoeICP-CLI/
cd MoeICP-CLI/
python3 cli.py
```

# 用法

```bash
    ___  ___          _____ _____ ______       _____  _     _____
    |  \/  |         |_   _/  __ \| ___ \     /  __ \| |   |_   _|
    | .  . | ___   ___ | | | /  \/| |_/ /_____| /  \/| |     | |
    | |\/| |/ _ \ / _ \| | | |    |  __/______| |    | |     | |
    | |  | | (_) |  __/| |_| \__/\| |         | \__/\| |_____| |_
    \_|  |_/\___/ \___\___/ \____/\_|          \____/\_____/\___/

Usage of cli.py:
cli.py [参数 萌备或者域名]
  help    查看帮助
  version 查看当前版本
  name    查询萌备名称
  domain  查询萌备域名
  desc    查询萌备描述
  home    查询萌备主页
  owner   查询萌备所有者
  uptime  查询萌备更新时间
  status  查询萌备状态
  number  查询萌备萌号
  pic     获取萌备图片链接

例子:
     cli.py info
     cli.py info 20230000
     cli.py help
     cli.py version
```
